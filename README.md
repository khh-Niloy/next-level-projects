# RaizMart - Trusted Marketplace for Premium Tech & Lifestyle

RaizMart is a premium tech and lifestyle e-commerce platform featuring **product variants, real-time search, cart, wishlist,** and a **full admin dashboard to manage products, orders, categories, blogs, and marketing tools** ‚Äî all secured with role-based access control.

E-commerce businesses struggle with **complex product catalogs and manual variant creation**. RaizMart provides an attribute-based product management system that automatically generates variants, COD payments, **a centralized admin dashboard**, real-time search, and integrated marketing tools. This delivers a scalable e-commerce solution that simplifies operations and improves customer experience.

# LocalLens - Where travelers connect with local experiences

LocalLens is a modern, role-based **tourism platform** consisting of a **Next.js frontend** and a **Node.js/Express backend**. Together these repos implement a tour discovery & booking ecosystem where **tourists can explore and book tours, guides can manage listings, and admins oversee operations.**

Travelers struggle to discover and book reliable local tours in one place, while local guides lack an easy, professional way to list tours, manage bookings, and reach customers, resulting in fragmented experiences on both sides. LocalLens solves this by providing a single, role-based platform where tourists can discover, book, and review tours, guides can manage listings and bookings, and admins oversee operations, combining discovery, booking, payments, and management into one streamlined system.

## üõ†Ô∏è Tech Used

# üì¶ Parcel Delivery System Backend

The **Parcel Delivery API** is a secure, modular, and role-based backend system built with **Express.js** and **MongoDB** using **Mongoose**. Inspired by real-world delivery systems like **Pathao Courier** and **Sundarban**, this project manages parcel operations from creation to delivery tracking.

It supports four user roles‚Äî**Sender**, **Receiver**, **Delivery Agent**, and **Admin**‚Äîwith tailored capabilities such as creating parcels, confirming delivery, managing delivery assignments, and overseeing system-wide operations.

‚ú® A key feature is the **automated assignment of parcels to available delivery agents**, streamlining dispatch and ensuring efficient delivery management. The platform ensures **status tracking**, **authentication**, **authorization**, and **real-time status logs**, making it suitable for production-ready logistics services.

Node.js, Express.js, MongoDB (Mongoose), JWT, bcrypt, Zod, Custom Middleware, cookie-parser, SSLCommerz, Postman, Vercel, TypeScript, ESLint, dotenv
